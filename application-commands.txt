Application commands to run

1. create .env file(Initial setup only)
2. composer install or composer update (Initial setup only)
3. database name and credentials adding in .env
4. Application key generate (Initial setup only)
    php artisan key:generate
5. php artisan config:clear
6. php artisan cache:clear
7. php artisan migrate(Initial setup only)
8. php artisan db:seed (Initial setup only)
9. php artisan serve(application run)
